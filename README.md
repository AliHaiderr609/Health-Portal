# Health Portal

Web client for **Health Portal** — a React application for users and therapists to manage appointments, profiles, and payments.

## Prerequisites

- [Node.js](https://nodejs.org/) (LTS recommended)
- npm

## Setup

```bash
npm install
```

## Run locally

```bash
npm start
```

Opens [http://localhost:3000](http://localhost:3000) in the browser.

## Build for production

```bash
npm run build
```

Output is written to the `build` folder.

## Test

```bash
npm test
```

## Tech stack

- React 18
- React Router, React Hook Form, Yup
- Bootstrap / MDB React UI Kit
- Axios, Stripe (payments)
